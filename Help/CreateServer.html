Create Server

What is Create Server?

The Create Server feature allows you to generate an executable file that, when run on a machine within the same network, makes it capable of receiving commands and executing processes via the scheduler. This is especially useful for automating tasks and managing distributed systems without manual intervention.

How Does It Work?

  • Executable Generation: Create Server generates an executable file that must be run on the target machine.
  • Pre-Configuration: The machine must be pre-configured with Python installed to run the scripts that the scheduler will point to at the scheduled times.
  • Task Execution: Once configured, the machine will be ready to handle automated tasks, respond to commands, and maintain continuous operation within your network.
Click on Create Server and wait for the loading to complete. Once finished, it will indicate the folder where the terminal server was created. Copy the folder to the desired computer and execute it. Now you have a service machine that will execute tasks and receive commands.

Benefits

  • Automation Without Intervention: Reduces the need for manual intervention, allowing tasks to be executed automatically.
  • Efficient Management: Facilitates the management of distributed systems, ensuring that all machines are ready to execute commands and processes.
  • Minimization of Downtime: Proactive configuration minimizes downtime and optimizes the operation of the server infrastructure.
