Scheduling Page

What is the Scheduling?

The Pypa scheduling page is designed to manage and schedule automation tasks in an organized manner. This interface allows users to set times, processes, and work queues to ensure tasks are executed as planned.

Features

  • Schedule Name: Field where the user enters the schedule name. Example: 'Example Task'.
  • Process: Selection of the process file to be executed. Example: 'Example_Process.py'.
  • Workqueue: Selection of the work queue associated with the process. Example: 'Example_Workqueue'.
  • Objects: Selection of objects required for the process.
  • Scheduled Time: Definition of the time at which the process should be executed.
  • Server: Selection of the server where the process will be executed. Example: 'DESKTOP-FG9THC4'.
  • Dates: Selection of the dates on which the schedule should occur.

Benefits

  • Organization: Allows you to manage and schedule tasks in an organized way, ensuring they are executed as planned.
  • Efficiency: Facilitates the definition of times and processes, optimizing task execution.
  • Flexibility: Allows you to adjust schedules and processes as needed.
  • Control: Ensures tasks are executed on the correct server at the defined time.

Practical Example

Imagine you need to run a data backup process every night at 11:00 PM. The Pypa scheduling page allows you to set this time and select the backup process, ensuring the task is automatically executed at the specified time.
